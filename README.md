# CareerMate_AI

CareerMate_AI is an agent-powered assistant for career guidance using LangChain and Ollama.

## Setup

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Start Ollama:
   ```
   ollama serve
   ```
3. Run the agent:
   ```
   python run_agent.py
   ```

## Features

- Skill gap analysis
- Job recommendations
- Course suggestions

## Requirements

- Python 3.8+
- Ollama installed and running
